# 📐 단위 변환기 앱
- `ConstraintLayout`을 이용하여 화면 그리기
- `ViewBinding`을 이용해 뷰에 대하여 접근
- `savedInstanceState`를 이용한 데이터 저장 및 복구  
<br>

## 구현 기능
1. `cm`를 `m`로 변환
2. 값을 입력하면, 바로 변환된 값이 노출 (입력값은 자연수로 한정)
3. 단위를 반대로 변경
4. 단위 변환 연산
    1. `cm` → `m` (x 0.01) 
    2. `m` → `cm` (x 100) 
<br>
